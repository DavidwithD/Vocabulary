# Vocabulary Builder

A Chrome extension that helps users build vocabulary in **English**, **French**, and **Spanish** by maintaining a personal word list per language and highlighting unfamiliar words (in all their forms) on any web page.

## Quick Start

1. Go to `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this project folder
4. Click the extension icon — select your language (English, Français, or Español) and CEFR level
5. Browse any web page — unfamiliar words are highlighted in lavender, learning words in amber
